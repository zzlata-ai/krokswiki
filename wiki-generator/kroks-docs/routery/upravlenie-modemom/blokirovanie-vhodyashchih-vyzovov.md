# Блокирование входящих вызовов

![](uploads/blokirovanie/block2.png)

При использовании сим-карты в модеме (роутере) работа 4G будет прервана, если на сим-карту поступит входящий вызов. Чтобы этого не происходило, для блокирования всех входящих вызовов можно использовать последовательность из трёх АТ-команд:

```bash
AT+CLCK="AI",1,"0000",1
AT+CLCK="AI",1,"0000",16
AT+CLCK="AI",1,"0000",32
```

где 0000 это PIN-код. PIN-код необходимо ввести тот, который был указан на сим-карте при её покупке (если позже он не был изменён).

Команды следует вводить в терминале, расположеном в Модем - Терминал.

:::tip
Обратите внимание, что данный метод может работать не на всех модемах и не со всеми операторами.
:::

Для разблокирования входящих вызовов, необходимо изменить "1" на "0" во всех ранее введённых командах.

```bash
AT+CLCK="AI",0,"0000",1
AT+CLCK="AI",0,"0000",16
AT+CLCK="AI",0,"0000",32
```
