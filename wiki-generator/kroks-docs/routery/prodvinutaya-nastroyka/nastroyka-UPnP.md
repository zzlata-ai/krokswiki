# Настройка UPnP

**UPnP** (Universal Plug and Play) — это протокол, который позволяет устройствам в домашней или локальной сети автоматически обнаруживать друг друга и взаимодействовать без необходимости сложной настройки. Он используется для упрощения подключения и обмена данными между различными устройствами, такими как компьютеры, принтеры, смартфоны, игровые консоли, маршрутизаторы и другие сетевые устройства.

## Основные назначения UPnP:

* **Автоматическая настройка сети**: Устройства могут автоматически определить друг друга и настроить необходимые параметры без вмешательства пользователя.
* **Обеспечение взаимодействия приложений**: Например, программы для потокового воспроизведения мультимедиа могут автоматически обнаруживать и подключаться к совместимым устройствам, таким как медиасервера или ТВ.
* **Настройка проброса портов**: UPnP позволяет устройствам автоматически открывать необходимые порты на маршрутизаторе для обеспечения работы сервисов, например, для игр, видеоконференций или P2P-сервисов.
* **Облегчение задач для пользователя**: Обеспечивает удобство и автоматизацию в настройке сети, особенно в домашних условиях, где пользователи могут не иметь специальных знаний в сетевых технологиях.

В целом, UPnP нужен для упрощения установки и эксплуатации сетевых устройств и сервисов, повышения их совместимости и автоматизации процессов.

## Настройка UPnP на роутерах KROKS

### Установка пакетов

По инструкции по установке пакетов устанавливаем luc-app-upnp. Все дополнительные пакеты установятся автоматически. После необходимо лишь выйти и заного войти в веб-интерфейс. Это необходимо для того чтобы в меню роутера отобразился пункт настройка UPnP.

::: info
Для установки соответствующих пакетов вручную необходимо подключиться к роутеру по ssh и выполнить следующие команды:
```bash
opkg update
opkg install miniupnpd luci-app-upnp
/etc/init.d/miniupnpd enable
/etc/init.d/miniupnpd start
/etc/init.d/firewall restart
```

С этого момента серсис UPnP запущен и работает в автоматическом режиме.
