# Обновление прошивки приборов Arinst

**Инструкция применима к следующим приборам:**

* Arinst VR 23-6200
* Arinst VR 1-6200
* Arinst VNA-PR1
* Arinst VNA-PR1-M
* Arinst VNA-DL
* Arinst SDR Dreamkit V1D
* Arinst SDR Dreamkit V2D
* Arinst ArSiG-R
* Arinst ArSiG-S
* Arinst SSA R3 (TG, LC, PRO, и пр.)
* Arinst BugHunter
* Arinst Signal Hunter
* Arinst SSA R2S

1. Скачайте и установите Arinst Virtual Lab с [сайта](https://arinst.ru/download-apk.php). Подключите устройство во включенном состоянии по USB.  
  ![Ссылки на сайте](uploads/5d6e4566-02ec-48bb-b117-12304ce8bfc0/c67a2df5-ba93-4e7a-9f9d-55042f43b533/97fefeba-1d81-4f5d-9859-39cfe70dc36b.png)

2. На главной странице ПО Arinst Virtual Lab нажмите кнопку подключения.  
  ![Главное окно](uploads/5d6e4566-02ec-48bb-b117-12304ce8bfc0/497e908a-ed20-40e5-8c95-11bfe2bed3f5/fe562353-0d6c-40e1-8dc0-927dead7a0a4.png)

3. В появившемся окне выберите устройство из списка.  
  ![Список устройств](uploads/5d6e4566-02ec-48bb-b117-12304ce8bfc0/306b5f42-801a-4422-be59-15bd4f74971b/1e6670db-fd11-46da-936d-354c3909eb07.png)

4. Если для устройства есть обновление и компьютер подключен к интернету, то обновиться можно согласившись во всплывшем окне.  
 ![Всплывающее окно](uploads/5d6e4566-02ec-48bb-b117-12304ce8bfc0/b100e7b3-ff11-4a35-b4bd-2d23c20c923f/10d5a628-3591-4337-b7e1-cb89282472bf.png)

4.1 Вручную проверить обновления и начать процесс можно на странице Device в меню приложения.  
 ![Меню Device](uploads/5d6e4566-02ec-48bb-b117-12304ce8bfc0/fbaa0fef-b0b6-4917-be7d-c2c494c2ae9a/bb23a6f8-d3eb-4788-ac52-1b1f497aed62.png)

::: danger
Если в процессе обновления появилось окно, как на примере ниже, то для продолжения процесса необходимо снова нажать на клавишу подключения и выбрать устройство.  
  ![Всплывающее окно](uploads/5d6e4566-02ec-48bb-b117-12304ce8bfc0/a570e1f9-fbae-4634-8a08-cb0c7d0fa34f/19d9b83f-649f-4aa1-aab1-b7dc803bd468.png)

:::
